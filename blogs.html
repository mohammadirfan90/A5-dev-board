Question 1: What are the different ways to select an element in the DOM?


We use getElementById("id"), getElementsByClassName("class"), getElementsByTagName("tag"), querySelector("selector"), querySelectorAll("selector") to select an element in DOM.

Question 2: What is the difference between innerHTML, innerText, and textContent?


For a given node, innerHTML contains HTML tags and text inside tags. innerText contains only the text inside tag that is displayed. textContent is like innerText but it will text even if text is hidden.

Question 3: What is event delegation in the DOM?


Event delegation is a method where event listener is added to a parent element, child elements events are handled using event.target.

Question 5: How do you create, add, and remove elements using JavaScript?


i. We create element using document.createElement().For example:
const newDiv = document.createElement("div");
ii. We add element to a parent node using document.parentNode.appendChild(newDiv); For example:
document.body.appendChild(newDiv);
iii. We remove elements using newDiv.remove(); or parentElement.removeChild(childElement);
